# StartWeChat

可以在电脑上启动多少微信

![startWeChat.jpg](startWeChat.jpg)

# 使用说明

无需安装，直接运行StartWeChat.exe
如果提示安装FrameWork，请访问[ https://dotnet.microsoft.com/en-us/download/visual-studio-sdks ](https://dotnet.microsoft.com/en-us/download/visual-studio-sdks)下载 .NET 5.0 进行安装后再运行程序
如果无法获取微信路径，请右键微信桌面图标，查看属性里微信的完整路径，如：D:\Program Files (x86)\Tencent\WeChat\WeChat.exe
